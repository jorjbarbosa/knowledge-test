## Knowledge level test

A simple RESTFUL API made with PHP, to get the API working you must have a APACHE server

Routes:

- Create  
  -- /api/user/create.php  
  -- Method: POST  
  -- input: {user: "Test", role: "test"}  
- Read  
  -- /api/user/read.php  
  -- Method: GET  
  -- input: none  
- Update  
  -- /api/user/delete.php  
  -- Method: PUT  
  -- input: {id: "some_id", user: "Test", role: "test"}  
- Delete  
  -- /api/user/update.php  
  -- Method: DELETE  
  -- input: {id: "some_id"}
